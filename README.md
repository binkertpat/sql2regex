# SQL2REGEX - the project

## scss → css
hard compile:
<code>npm run scss</code>

compile changes on runtime:
<code>npm run watch</code>

## install/build application
<code>./mvnw clean install</code>

## start application
<code>./mvnw spring-boot:run </code>
